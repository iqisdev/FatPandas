# FatPandas

## About

FatPandas is an Android-based food ordering application developed using Java in Android Studio. The application allows users to browse food menus, add items to favorites, manage a shopping cart, and place orders.

## Features

- User-friendly interface
- Browse food menu
- Add and remove favorite items
- Shopping cart management
- Update item quantity in cart
- Checkout process
- Order history
- Persistent data storage using SQLite

## Technologies

- Java
- Android Studio
- SQLite
- XML

## Getting Started

1. Clone this repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Run the application on an emulator or Android device.

## Improvements

- Implemented SQLite database for persistent storage.
- Cart data is saved even after the application is closed.
- Favorite items are stored permanently.
- Order history is saved locally.
- Improved cart quantity update and checkout process.
